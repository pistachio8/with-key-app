📢 with-key 업데이트 (5/28)

오늘 12건 반영됐어요. 앱 새로고침(또는 PWA 재실행) 한 번 해주세요!

✨ 새 기능

- 어디서나 빠른 메뉴 — 화면 하단 가운데 동그란 버튼을 누르면 홈·사진 인증·그룹 메뉴가 부채꼴로 펼쳐져요. "사진 인증"은 지금 보고 있는 챌린지를 알아서 골라줍니다. (챌린지 밖이면 진행 중인 챌린지로 바로, 여러 개면 선택 화면이 떠요.)
- 일기 직접 입력 모드 — AI 생성 대신 직접 쓰고 싶으면 메모에 글을 적으면 그대로 저장돼요. 이때는 키워드를 고르지 않아도 제출할 수 있어요.
- 식사 인증 일기 개선 — 끼니 인증 시 올린 시간대(아침·점심·저녁·야식)를 반영한 일기 문구가 나와요.
- 인증한 시각 표시 — 인증 피드에서 각 인증이 "방금 전 / N분 전 / N시간 전"으로 언제 올라왔는지 보여요. 하루가 지나면 날짜로 표시됩니다.
- 홈 "예정 벌금"이 내 기준으로 — 그룹 전체 최대 금액 대신, 내가 내야 할 예정 벌금을 보여줘요. 이번 주 목표를 채우면 0원으로 내려갑니다.
- 서약서 도장 디자인 — 서약서 카드 우상단에 from·with 원형 도장이 추가됐어요.

🐛 버그 수정

- 피드 사진 안 뜨던 현상 — 홈에 들어가거나 앱을 백그라운드에 갔다가 다시 열 때 인증 사진이 안 보이던 문제를 고쳤어요.
- 아이폰 계좌번호 복사 — 정산 계좌 시트에서 "복사에 실패했어요"가 뜨던 문제를 수정했어요 (iOS Safari·홈화면 앱).
- 정보탭 "모인 예정 벌금" 정정 — 아직 벌금이 쌓이지 않았는데도 부풀려 보이던 금액을 실제 정산 합계로 바로잡았어요.
- 잘못된 링크 진입 — 없는 페이지나 해산된 그룹 링크로 들어가면 깔끔한 안내 화면 + "홈으로 가기" 버튼이 나와요.
