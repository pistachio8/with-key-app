📢 with-key 업데이트 (5/29)

오늘 7건 반영됐어요. 앱 새로고침(또는 PWA 재실행) 한 번 해주세요!

✨ 새 기능

- 인증 완료 화면이 화려해졌어요 — 인증을 마치면 그동안 인증한 날들이 달력에 색으로 채워지고, 연속으로 인증할수록 그 날 칸이 더 진하게 보여요. 이번 주 목표를 처음 달성하는 순간엔 축하 컨페티가 터집니다.
- 친구 인증 완료 알림 — 같은 그룹 친구가 사진 인증을 마치면 알림이 와요. 전에는 인증 화면에 들어가기만 해도 알림이 갔는데, 이제 실제로 완료했을 때 옵니다. 러닝·헬스·요가·식단 등 활동에 맞는 문구로 나와요. (알림 설정의 "시작 알림"은 "그룹 활동 알림"으로 이름이 바뀌었어요.)
- 정산 공유 영상 — 정산 결과를 공유할 때 그룹 인증 사진들이 흐르는 짧은 영상으로도 만들 수 있어요. 공유 화면에서 영상·사진·티켓 중 골라 보내세요.
- 정산 페이지가 영수증 카드로 — 정산(돌아보기) 페이지가 공유 카드와 같은 톤의 영수증 디자인으로 통일됐어요. 혼자 하는 챌린지면 사진 갤러리가 함께 보입니다.

🐛 버그 수정

- 인증 폼에 이전 내용이 남던 문제 — 사진·메모를 쓰다 다른 화면에 갔다가 다시 인증 화면으로 오면 이전에 쓰던 내용(사진 포함)이 그대로 남던 문제를 고쳤어요. 새로 시작할 땐 깨끗한 폼이 떠요.
- 공유 카드가 안 만들어지던 문제 — 정산 공유 카드(사진형·티켓형)를 만들 때 실패하던 문제를 수정했어요.

🔧 소소한 개선

- 공유 메시지 다듬기 — 정산을 공유할 때 메시지에 "from. with" 브랜드 표기와 서비스 홈 링크가 함께 붙어요.
