# DECISIONS.md — Skill-managed ADR Log

> 이 파일은 `architecture-decision-records` 스킬이 관리하는 결정 로그입니다.
> 팀 공유용 원본 로그는 [`TEAM_SHARE_DECISIONS.md`](./TEAM_SHARE_DECISIONS.md)에 있습니다.
>
> **이 파일의 역할**: 코딩 세션 중 발생하는 아키텍처 결정(기술 스택, 패턴, API/스키마, 라이브러리 선택 등)을 Nygard ADR 포맷으로 즉시 기록.
> **팀 공유가 필요한 결정**은 이 파일이 아니라 `TEAM_SHARE_DECISIONS.md`에 추가하세요.
>
> **원칙**
> - 최신이 위. 번호 ADR-NNNN 단순 증가 (ADR-0001부터).
> - 엔트리 **수정 금지** — 뒤집으려면 새 엔트리로 `superseded by ADR-NNNN` 표시.
> - 사소한 결정(변수명, 포맷팅) 기록 안 함.
> - 반드시 **거부된 대안**과 **그 이유**를 남긴다.
