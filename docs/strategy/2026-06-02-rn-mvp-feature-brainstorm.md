## 0. 전제 — 포팅으로 그대로 가져가는 POC 5종

아래는 "신기능 후보"가 아니라 **이미 검증된 핵심 루프**다. RN 포팅 대상이며 이 아이디에이션의 평가 대상에서 제외한다.

| #   | 기능                          | 현재 POC 상태                              |
| --- | ----------------------------- | ------------------------------------------ |
| 1   | 그룹 서약서(전원 서명 → 시작) | 완성                                       |
| 2   | 사진+키워드 인증              | 완성                                       |
| 3   | AI 운동일기 자동생성          | 완성 (gpt-4o-mini, 4.5s timeout, fallback) |
| 4   | 피드 / Kudos                  | 완성                                       |
| 5   | 정산 / recap                  | **"예정 벌금 표시만"** — 실제 정산 없음    |

> 핵심 공백: POC는 정산이 "표시만"이라 손실회피가 **연출**에 그친다. 신기능 후보의 대부분은 이 공백(정산을 실제 동기로)과 그 전제(자동 판정)를 메우는 데 모인다.

## 4. Top 5 추천

| 순위 | 아이디어                                        | 왜                                                                            | Quick Win?             |
| ---- | ----------------------------------------------- | ----------------------------------------------------------------------------- | ---------------------- |
| 1    | **P1 포인트 정산 (closed-loop)**                | 비즈니스 모델의 심장. POC "표시만"을 실제 손실회피·0% rake 신뢰 엔진으로 전환 | ✗ 전략 bet             |
| 2    | **P2 사진 자동검증**                            | 정산 자동화·분쟁 제거의 **전제**. 이미 discovery 완료 → P1과 한 쌍            | ✗ 전략 bet (P1과 페어) |
| 3    | **D1+E1 네이티브 카메라 + 온디바이스 사전검증** | 자동검증 신뢰를 끌어올리는 enabling 요소. RN 포팅에 자연 포함                 | ✓ Quick win            |
| 4    | **D2 보증금 시각화**                            | 손실회피 감정 설계. 저비용·고효과                                             | ✓ Quick win            |
| 5    | **E3 anti-cheat 중복검출**                      | 자동검증·포인트 신뢰의 방어선. P1·P2가 깨지지 않게 보호                       | △ 보조 bet             |

### 핵심 통찰 — "신기능 1~2개"의 정체는 한 쌍이다

**P1(포인트 정산)과 P2(사진 자동검증)는 분리할 수 없는 한 쌍**이다.

- 자동검증(P2)이 없으면 정산(P1)은 그룹장 수동 판정에 의존 → 분쟁·운영 부담으로 무너진다.
- 정산(P1)이 없으면 자동검증(P2)은 "표시만"을 자동화할 뿐 동기 엔진이 안 된다.
- 둘이 합쳐져야 POC의 연출용 벌금이 **실제 손실회피 + 0% rake 신뢰 차별점**(전략의 핵심 주장)이 된다.

→ **권장 MVP 신기능 = P1 + P2 (한 쌍)**, D1·E1·E3는 이 쌍을 성립시키는 enabling/방어 요소로 포팅 스코프에 흡수.
→ **MVP 보류**: P3 구독(WTP 약함, 런치 후), P4 B2B 대시보드(2순위), D3 위젯·E5 Health(폴리시).
