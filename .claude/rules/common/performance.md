# 성능 최적화

> 비용·컨텍스트 운영의 원본은 [`../../../docs/QUALITY_GATE.md`](../../../docs/QUALITY_GATE.md) "AI 에이전트 비용·컨텍스트 운영". 이 파일은 Claude/ECC 세션 실행 기준만 추가.

## 컨텍스트 윈도우 마지막 20%에서 피할 것

- 대규모 리팩토링
- 여러 파일에 걸친 기능 구현
- 복잡한 상호작용 디버깅

(`/compact` 시점은 QUALITY_GATE 기준 따름.)

## 확장 사고 + 계획 모드

확장 사고는 기본 활성, 내부 추론에 최대 31,999 토큰.

- 전환: Option+T (mac) / Alt+T (Win/Linux)
- 설정: `~/.claude/settings.json`의 `alwaysThinkingEnabled`
- 예산: `export MAX_THINKING_TOKENS=10000`
- 출력 확인: Ctrl+O

깊은 추론이 필요하면: 확장 사고 활성 → 계획 모드 → 여러 라운드 비판 → 역할 분리 서브에이전트.

## 빌드 실패 시

빌드 에러는 에러 메시지 분석 → 점진적 수정 → 각 수정 후 재검증 순으로 처리한다. 범위가 넓으면 general-purpose 서브에이전트에 위임한다.
